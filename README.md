# Frontend Technical Assessment

React Flow frontend + FastAPI backend for pipeline creation and DAG validation.

## Quick Start

1. Start backend: `cd backend` then `uvicorn main:app --reload --port 8000`
2. Start frontend: `cd frontend` then `npm install && npm start`

Frontend: http://localhost:3000  
Backend: http://127.0.0.1:8000
